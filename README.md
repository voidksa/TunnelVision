<p align="center">
  <b>Tunnel Vision</b> is a lightweight, portable Windows utility designed to eliminate visual noise. It automatically dims your screen while keeping the active window in the spotlight, helping you maintain deep focus during work or gaming sessions.
</p>

## üì• Download & Installation

1. **Download** the latest `TunnelVision.zip` from the [Releases Page](https://github.com/voidksa/TunnelVision/releases/latest).
2. **Extract** the contents to a folder of your choice (e.g., `Documents` or `Desktop`).
3. **Run** `TunnelVision.exe`.

> **Note:** Tunnel Vision requires [.NET Desktop Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0). If not installed, Windows will prompt you to download it.

## üéÆ Usage
